java里面也有这种情况:

一般格式就是: var 变量 = 表达式1?表达式2: 表达式3;

执行过程:

*  就看表达式1的结果是true还是false了.
*  如果是true则执行表达式2,然后把结果给变量
*  如果表达式1的结果是false,则执行表达式3,把结果给变量
